# Markflow

Markflow is an Obsidian plugin that lets you draw lightweight diagrams and sketches directly inside Markdown notes.

Itâ€™s designed to be simple, markdown-native.

## What Markflow is
- An inline drawing block for Markdown.
- Stores drawing data inside the Markdown file.
- Optimized for diagrams, flows, and quick sketches.


## Development status
Markflow is under active development.
Expect breaking changes while the core format stabilizes.


## How to use
- Clone this repo.
- Make sure your NodeJS is at least v16 (`node --version`).
- `npm i` or `yarn` to install dependencies.
- `npm run dev` to start compilation in watch mode.

## Manually installing the plugin

- Copy over `main.js`, `styles.css`, `manifest.json` to your vault `VaultFolder/.obsidian/plugins/your-plugin-id/`.
